# ACM onboarding

This is the onboarding system used to enter at the American Corner of Mahajanga.

## Installation

-   Clone the github repo: `git clone https://github.com/Ianel/acm-check-in.git`
-   Install the dependencies: `npm install`
-   Start the server: `npm run dev`

## Tech stack

-   React 19 + Typescript
-   Tailwind CSS + Shadcn UI
-   Vite
